# Mentored junior developers to improve team velocity

**Situation:** Our team had 3 new junior developers who were struggling with the codebase, causing our sprint velocity to drop by 30%.

**Task:** As the senior engineer on the team, I needed to get them up to speed while maintaining delivery timelines.

**Action:** I designed a structured onboarding program:
- Weekly pair programming sessions focused on different parts of the codebase
- Created comprehensive documentation and runbooks
- Set up a mentorship rotation so each junior worked with different seniors
- Established code review guidelines with constructive feedback

**Result:** Within 3 months, all 3 juniors were independently handling tickets. Team velocity recovered and exceeded previous levels by 15%. The onboarding program became the team standard.
